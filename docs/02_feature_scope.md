# 02. 機能一覧（MVP スコープ）

## 公開側（入れる機能）
- 記事一覧（ページネーション）
- 記事詳細（Markdown レンダリング）

## 公開側（今回は入れない）
- タグ一覧ページ（`/tags`）
- カテゴリ一覧ページ（`/categories`）
- タグフィルタ
- カテゴリフィルタ
- フリーワード検索
- コメント投稿
- SNS共有リンク
- OGP（SNS連携がないため後回し）
- 同カテゴリ/同タグのおすすめ記事
---

## 管理側（入れる機能）
- GitHub OAuth（NextAuth）によるログイン
- 記事一覧（下書き／公開ステータス）
- 記事作成（Markdown 入力＋プレビュー）
- 下書きバッジ表示
- 公開 / 下書きのタブ切り替え
- 記事編集
- 下書き機能（isDraft）
- タグ／カテゴリの設定（記事作成・編集時）

## 管理側（今回は入れない）
- コメント管理（コメント機能を非採用）
- 画像アップロード
- 記事削除

---

## バックエンド（入れる機能）
- 記事 CRUD API
- ページネーション API
- 認証チェック（NextAuth のセッション/JWT を検証）
- isDraft による公開/下書きの切替

---

## 対象外（後回し）
- タグ一覧・カテゴリ一覧の公開導線
- タグ／カテゴリによる記事フィルタ
- コメント機能全般
- 記事本文向け画像アップロード
- 記事の物理削除
- 全文検索（タイトル・本文検索）
- SNS共有機能
- 高度な OGP 設定
- 複数ユーザー管理
- 通知機能
